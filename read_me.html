Decimal Operations

by Luan Valle

The Decimal structure is a C structure that allows you to work with fixed-point decimals (9 decimal digits and 9 integer digits). To start using it, you need to include the header file `decimal.h` in your C code.

#include "decimal.h" 
Or
#include "decimal.c"

Available Functions

The Decimal structure provides the following functions:

Function Description
set_value(Decimal *this, char *number) Sets the value of the decimal number.
get_number(Decimal *this) Returns the decimal number as a formatted string.
decimal_add(Decimal *res, Decimal *n1, Decimal *n2) Adds two decimal numbers.
decimal_sub(Decimal *res, Decimal *n1, Decimal *n2) Subtracts two decimal numbers.
decimal_mult(Decimal *res, Decimal *n1, Decimal *n2) Multiplies two decimal numbers.

Example of Use

Here's an example of how to use the Decimal structure and its operation functions:

#include "decimal.h"

int main() {
    // Create new decimal numbers
    Decimal *num1 = new_Decimal();
    Decimal *num2 = new_Decimal();
    Decimal *result = new_Decimal();
